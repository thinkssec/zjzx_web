Pivot table

Two dimensional grouping of plain data into pivot table

Example creates two dimensional pivot table from plain data. Click to "Show master" button to see the source data.
In top row there are columns available as pivot sources. Drag them to the second row to set up the pivot columns, rows and data and click to Create pivot button.
In the third row there are buttons for creating, clearing and switching the pivot table. You can also choose some from predefined pivot settings from list to set up the pivot table.
You can modify the pivot or source table (edit cells, add, delete and move rows) and the second grid will be automatically updated accordingly. In real application the changes can be also saved to original database.
Colors in the pivot table: White background with blue font are the original data, they can be edited if they represent only one source row (not summary). Red background are immediate summaries, the others are summary summaries.
Too large tables, with too many columns, don't permit expanding all columns, but only one column per higher level. To speed up the grid.
